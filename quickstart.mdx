## 安装 Gradio

首先，使用 pip 安装 Gradio：

```bash
pip install gradio
```

## 第一个 Gradio 应用

### 文本生成演示

```python
import gradio as gr

def generate_text(prompt):
    # 这里是一个简单的文本生成示例
    # 在实际应用中，您可以替换为真实的 AI 模型
    return f"生成的文本: {prompt}"

demo = gr.Interface(
    fn=generate_text,
    inputs=gr.Textbox(label="输入提示词"),
    outputs=gr.Textbox(label="生成结果")
)

demo.launch()
```

## 不同类型的界面

### 图像分类

```python
import gradio as gr
import numpy as np
from PIL import Image

def classify_image(image):
    # 模拟图像分类
    # 实际应用中应使用真实的机器学习模型
    classes = {
        "猫": 0.7, 
        "狗": 0.3
    }
    return classes

demo = gr.Interface(
    fn=classify_image,
    inputs=gr.Image(type="pil", label="上传图像"),
    outputs=gr.Label(label="分类结果")
)

demo.launch()
```

## 高级配置

### 多输入/输出界面

```python
import gradio as gr

def process_data(text, number):
    return {
        "处理后的文本": text.upper(),
        "数字平方": number ** 2
    }

demo = gr.Interface(
    fn=process_data,
    inputs=[
        gr.Textbox(label="输入文本"),
        gr.Number(label="输入数字")
    ],
    outputs=[
        gr.Textbox(label="大写文本"),
        gr.Number(label="数字平方")
    ]
)

demo.launch()
```

## 常见界面类型

1. `gr.Interface()`: 最简单的界面
2. `gr.Blocks()`: 更复杂的自定义布局
3. `gr.TabbedInterface()`: 多标签界面

## 部署与分享

- 使用 `share=True` 生成临时公共链接
- 集成 Hugging Face Spaces
- 自定义域名部署

```python
demo.launch(share=True)  # 生成临时链接
```
